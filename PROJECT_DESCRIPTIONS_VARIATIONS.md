## 📖 Medium Description (For Railway Project/README Badge)

**CRACLE (Cognitive Reasoning and Adaptive Learning Engine)** is an AI-powered educational platform that uses 7 specialized agents (Memory, Planner, Content Generator, Simulation, Evaluator, Mentor, Orchestrator) to deliver truly personalized learning experiences. 

Built with FastAPI + React + Azure OpenAI (GPT-4o), CRACLE features:
- Adaptive learning paths that evolve with you
- Real-time AI reasoning visualization via WebSocket
- On-demand content generation for any topic
- Interactive decision simulations
- Cognitive profile tracking and evolution
- Personal AI mentor with context awareness

Perfect for self-learners, educators, and organizations seeking scalable, intelligent education technology.
